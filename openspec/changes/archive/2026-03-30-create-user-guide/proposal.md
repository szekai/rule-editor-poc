## Why

The SpEL Rule Editor is a complex application with multiple features (rule editing, rule set composition with drag-and-drop, maker/checker workflow, SpEL validation). New users need a comprehensive guide to understand how to use all features effectively. Currently, only a basic README exists, which is insufficient for onboarding users and demonstrating the full capabilities of the application.

## What Changes

- Create a comprehensive user guide documentation in the `docs/` directory
- Document all three main features: Rule Editor, Rule Set Composer, and Maker/Checker
- Include step-by-step tutorials for common workflows
- Add visual aids (ASCII diagrams, screenshots descriptions)
- Cover configuration and environment variables
- Include troubleshooting section for common issues

## Capabilities

### New Capabilities
- `user-guide`: A comprehensive user guide covering all application features, workflows, and configurations

### Modified Capabilities
- None - this is new documentation only

## Impact

- New `docs/` directory structure for user guide content
- May include updates to existing README.md to link to the comprehensive guide
- No code changes required
